A simulation model may have an underlying space model. While in the real world, we live in a three-dimensional continuous space, a simulation model can be based on a simplified space model such as

  1. a two-dimensional continuous space,
  2. a two-dimensional discrete space, also called grid space,
  3. a one-dimensional continuous, or discrete, space,
  4. a graph space consisting of a set of locations, which may, or may not, be connected with each other.

OESjs 1.4 only supports grid space, but other space models will be supported in future versions. For defining a grid space model in simulation.js, the following settings have to be made:

sim.model.space.type = "IntegerGrid";
sim.model.space.xMax = 500;
sim.model.space.yMax = 300;

The OESjs Integer Grid space model allows large grid spaces with large populations. An example of a grid space model using Integer Grid is the Schelling Segregation Model.
