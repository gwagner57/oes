The Pizza-Service-1 model is a single server queuing system model consisting of the object type "PizzaService", the event type "Order", and the simple activity type "MakePizza", which is not explicitly modeled as a resource-constrained activity type.

Default scenario

The default scenario has a single pizza service company.
