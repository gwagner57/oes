OES JavaScript Core 4a Simulator

The OESjs Core 4a simulator implements an architecture for Object Event Simulation (OES) that extends the OESjs Core 3 simulator by adding support for Agents.

You can download and inspect the code from the OES GitHub repo.

Simulation Examples

  • Four-Stage-Supply-Chain-1

    A four stage supply chain consists of a retailer, a wholesaler, a distributor and a factory.

  • Signaling-Game-1

    A simulation of a signaling reinforcement learning (RL) process, where two agents learn to communicate with each other via signals.
