OES JavaScript Core 2 Simulator

The OESjs Core 2 simulator implements an architecture for Object Event Simulation (OES) that extends the OESjs Core 1 simulator by adding support for activities, which may be resource-constrained.

The activities extension includes the following additional code (files): (1) OES-Activities.js in OESjs-Core2, (2) the sim.stat.initialize function in OESjs-simulator.js, (3) an extension of the oes.ui.showStatistics function in OESjs-simulatorUI.js
