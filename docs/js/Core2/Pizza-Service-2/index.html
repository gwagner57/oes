The Pizza-Service-2 model is a single server queuing system model consisting of the object type PizzaService, the event type Order, and the resource-constrained activity type MakePizza, which has the resource role pizzaService expressing that a MakePizza activity requires a PizzaService company as a resource.

Default scenario

The default scenario has a single pizza service company.
